Quick reference

planet

Declares a variable and assigns it the result of an expression or a string.

planet fuel = 80 + 5;
dock

Prints a single value — a number, a string, or a variable — to the mission log.

dock fuel;
emit

Ends the program, optionally with an exit code expression.

emit 0;
" … "

String literals. Only one is read between quotes — no escapes yet.

dock "all systems go";
